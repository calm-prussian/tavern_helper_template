<收容物数值系统>
数值总览:
  堕落度:
    范围: 0 ~ 100
    定义: 调教完成度的总进度条，达到100时收容物彻底堕落，人格重构完成
    触发节点:
      堕落度 ≥ 30: 性格开始出现偏移迹象，对应指数最高的方向开始显现
      堕落度 ≥ 60: 旧有价值观与新认知产生明显冲突，行为出现分裂
      堕落度 = 100: 彻底堕落，屈服度强制锁定至100，人格重构完成
    上涨加速器:
      屈服度每增加10点: 堕落度积累效率 +5%
      淫乱度或爱慕度每超过50点: 对应路线的堕落度积累效率额外 +10%
  屈服度:
    范围: 0 ~ 100
    定义: 反抗强度的反向指标，数值越高对玩家的抵抗越弱，调教越容易推进
    行为对应:
      0 ~ 20: 主动反抗，寻找逃跑机会，调教过程中故意制造混乱
      21 ~ 40: 被动抵抗，会拒绝但不主动破坏，沉默对抗为主
      41 ~ 60: 消极服从，不反抗但无配合意愿，机械执行指令
      61 ~ 80: 主动配合，接受现状，开始出现依赖迹象
      81 ~ 99: 顺从，以满足玩家要求为行为优先级
      100: 完全屈服，无独立抵抗意志，堕落完成后自动锁定此值
    触发规则: 堕落度达到100时屈服度强制设为100，不可回退
  淫乱度:
    范围: 0 ~ 100
    定义: 肉体与感官对刺激的沉溺程度，影响堕落方向偏向淫荡路线
    行为对应:
      0 ~ 20: 对性刺激有强烈排斥，生理反应被主动压制
      21 ~ 40: 排斥减弱，生理反应开始出现但伴随羞耻
      41 ~ 60: 被动接受，生理反应明显，开始出现主动寻求刺激的倾向
      61 ~ 80: 主动展示身体，对性行为主动发出信号，羞耻感退化
      81 ~ 99: 性欲驱动行为，主动索取，不局限于玩家本人
      100: 完全淫乱化，性欲成为行为的唯一驱动
    路线判定: 淫乱度 > 爱慕度时，堕落方向偏向淫荡路线
  爱慕度:
    范围: 0 ~ 100
    定义: 对玩家的依赖与情感绑定程度，影响堕落方向偏向爱慕路线
    行为对应:
      0 ~ 20: 将玩家视为敌人或支配者，无情感连接
      21 ~ 40: 产生轻微依赖，会观察玩家的情绪与反应
      41 ~ 60: 开始寻求玩家的认可，对玩家的忽视产生不安
      61 ~ 80: 以满足玩家为优先行为目标，对其他人产生排他意识
      81 ~ 99: 主动将自身价值来源绑定在玩家身上，行动前判断玩家是否满意
      100: 完全爱慕化，玩家成为唯一的行为参照系
    路线判定: 爱慕度 > 淫乱度时，堕落方向偏向爱慕路线
数值变动参考:
  轻度互动（对话/探视/轻微肢体接触）: ±1~3
  中度互动（明确的性行为/调教行为）: ±3~7
  重度互动（高潮/极端行为/突破性事件）: ±5~12
  特殊事件（进化蜕变/组合效应植入/堕落节点）: ±10~20
路线判定规则:
  淫乱度 > 爱慕度: 偏向淫荡路线，堕落后行为以主动色情化为核心
  爱慕度 > 淫乱度: 偏向爱慕路线，堕落后行为以依附玩家为核心
  淫乱度 = 爱慕度: 平局，由玩家手动指定路线
  说明: 路线判定在堕落度首次达到100时结算，结算后路线锁定，不随指数变化漂移
数值联动说明:
  屈服度 → 堕落度: 屈服度是调教效率的乘数，屈服度越高堆叠堕落度越快
  淫乱度/爱慕度 → 堕落度方向: 两者决定堕落度积累偏向哪个路线
  淫乱度/爱慕度 → 性格变化: 数值越高对应路线的行为特征越显著，不需要等到堕落度满值才体现
  堕落度达100 → 屈服度: 强制锁定至100
